RSS & Atom feeds

22 Aug, 2025

Each space has an RSS and Atom feed for use in RSS readers.

Note: The /feed/ URL shows your drops list page in the browser. For actual RSS/Atom feeds, use the specific feed URLs below.

Atom feed

https://<your-space>.vibera.dev/feed/atom/

Eg: https://vibera.vibera.dev/feed/atom/

RSS feed

https://<your-space>.vibera.dev/feed/rss/

Eg: https://vibera.vibera.dev/feed/rss/

Alternative feed URLs

The following URLs also work for compatibility:

https://<your-space>.vibera.dev/atom/
https://<your-space>.vibera.dev/rss/
https://<your-space>.vibera.dev/feed.xml
https://<your-space>.vibera.dev/atom.xml
https://<your-space>.vibera.dev/rss.xml

Tag specific feeds

Feeds filtered by a tag can be accessed with the query parameter q

https://<your-space>.vibera.dev/feed/atom/?q=<tag-name>
https://<your-space>.vibera.dev/feed/rss/?q=<tag-name>

Eg: http://vibera.vibera.dev/feed/atom/?q=web-development
