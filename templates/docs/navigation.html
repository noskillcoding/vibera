The navigation section uses standard markdown for handling links.

The following is a set of 2 links, the first linking to the root / and the second linking to the feed:

To have the drops list as your homepage, leave the content (everything below the ___) of your homepage empty.

[Home](/) [Feed](/feed/)

Home Feed

To add a page as a link, set up the page in Drops and use the link as the link address:

[Home](/) [Projects](/projects/)

Home Projects

To link to your drops list page, you can use the space path:

[Home](/) [Space](/space/)

Home Space

Adding external links works similarly:

[Home](/) [Example](https://example.com)

Home Example

And finally, using custom vibera markdown syntax for adding external links:

[Home](/) [Example](tab:https://example.com)

Home Example

For linking to content on the same page (internal linking) you can specify the id of a heading or subheading as a hash-value. Eg: # Subsection 2 will automatically have an id of subsection-2. This can be linked to like this:

[Subsection 2 link](#subsection-2)
