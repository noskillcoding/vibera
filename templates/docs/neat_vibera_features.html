Vibera features

22 Aug, 2025

Here are some useful vibera features you might not know about.

Dashboard customization

You can customize your dashboard appearance in Account > Customise dashboard:

  • Dashboard styles - Custom CSS for your dashboard
  • Dashboard footer - Custom content for dashboard footer
  • Landing page styles - Custom CSS for the main vibera landing page
  • Discovery feed styles - Custom CSS for the discovery feed

Embedded drop lists

Add filtered drop lists to any content. This is covered in detail here.

Edit robots.txt

You can edit your robots.txt file in Settings > Advanced settings. For example, block AI crawlers:

User-agent: GPTBot
Disallow: /

User-agent: ChatGPT-User
Disallow: /

Media uploads

Upload images, videos, and other files directly to your drops:

  • Up to 15 files per drop
  • 10MB per file limit
  • Visual thumbnails with click-to-insert
  • Support for images, videos, audio, documents, and fonts

GitHub integration

Link your drops to their source code repositories using the GitHub URL field or frontmatter:

github_url: https://github.com/username/project-name

Tags and tools

Categorize your projects with tags and specify technologies used:

  • Popular suggestions based on platform usage
  • Used for filtering and discovery
  • Help other developers find relevant projects

Comments and reports

Built-in native comments and content reporting:

  • Users can comment with nickname, email, or anonymously
  • Report dangerous or inappropriate content
  • Self-moderation with user deletion options
