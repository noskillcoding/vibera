Themes and styling

22 Aug, 2025

Vibera has built-in style editor - so you can customize pretty much everything.

Themes

There are a handful of pre-built themes to choose from in the dashboard under the Themes section. These can be used as-is, but also make a good jumping off point for creating custom styles.

You can also choose any no-class css theme as a starting point and include it in the Edit theme CSS field using @import https://link-to-theme.css; (custom fonts can be imported in the same way).

Custom styles

Vibera's themes can be customised using pure CSS in the Edit theme CSS field. The structure of the theme CSS is set up to easily modify the basics such as font, colour, scale, etc (both for light mode and dark mode, as specified by the @media (prefers-color-scheme: dark) rule).

Dashboard customization

Vibera offers additional styling options for different parts of the platform:

  • Dashboard styles - Custom CSS for your personal dashboard
  • Dashboard footer - Custom content for dashboard footer
  • Landing page styles - Custom CSS for the main vibera landing page
  • Discovery feed styles - Custom CSS for the discovery feed

These can be configured in Account > Customise dashboard.

Vibera classes

Vibera has several classes which can be used as CSS selectors to customise specific pages or elements.

Pages

  • The root of the homepage: .home {}
  • The root of any drop: .post {}
  • The root of any page: .page {}
    • If page and drop styles are the same, both can be selected using .page .post {}
  • The root of the space list page: .blog {}
  • The root of the email subscription page: .subscribe {}
  • The root of the 404 not found page: .not-found {}

Custom class names

Each drop has the header option class_name which adds a class to the root of the drop to allow custom styling for that particular page.

Eg:

class_name: vibez

Elements

  • The anchor tag for the title of the space: .title h1 {}
  • The ul of the drops on the space: .blog-posts {}
  • The navigation header: nav
  • The codeblocks/highlights: .highlight, .code {}
  • The drop showcase cards: .drop-card {}
  • The drop showcase container: .drop-showcase {}

Notes

All parts of the front-end space are selectable and style-able using simple CSS selectors.

The "Powered by vibera" footer can be customized or hidden if desired, as all vibera users have full customization access.
