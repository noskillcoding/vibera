GitHub integration

22 Aug, 2025

Vibera includes built-in GitHub integration to link your project drops directly to their source code repositories.

Adding GitHub links

When creating or editing a drop, you can add a GitHub repository URL:

  • Use the dedicated GitHub URL field in the drop editor
  • Or add it via frontmatter in your drop content
  • Link appears prominently on your drop page

Frontmatter example

title: My Awesome Project
github_url: https://github.com/username/project-name
___
Project description goes here...

The GitHub URL field accepts any valid repository URL and displays it as a clickable link on your drop page, making it easy for other developers to explore your code.
