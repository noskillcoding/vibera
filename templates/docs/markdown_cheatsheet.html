Linebreaks

Adding a single line break will not be rendered as a line break.

Here's some text.
Here's some more text.

Here's some text. Here's some more text.

To add a single line break use a \ or two spaces at the end of the line.

Here's some text.\
Here's some more text.

Here's some text.
Here's some more text.

Adding 2 line breaks starts a new paragraph.

Here is one paragraph with lengthy text.

Here is another.

Here is one paragraph with lengthy text.

Here is another.

Footnotes

Here is a simple footnote,[^1] and here is a longer one.[^named-note]

Then at the bottom of your drop...

[^1]: This is the first footnote.

[^named-note]: Here is another one.

Here's a simple footnote,1 and here's a longer one.2

Project showcase tips

Structure your drops

A well-structured project drop typically includes:

  • Overview: Brief description of what the project does
  • Features: Key functionality and capabilities
  • Tech stack: Tools and technologies used
  • Installation/Usage: How to run or use the project
  • Screenshots/Demo: Visual representation of the project
  • Challenges: What you learned or overcame
