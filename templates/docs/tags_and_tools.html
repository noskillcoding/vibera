Tags and tools

22 Aug, 2025

Vibera uses tags and tools to categorize and make your project drops discoverable. These help other developers find projects that use similar technologies or solve similar problems.

Tags

Tags describe what your project is about or what category it fits into:

  • Examples: "web-app", "machine-learning", "game", "mobile", "cli-tool"
  • Help categorize your project's purpose or domain
  • Used for filtering and discovery

Tools

Tools specify the technologies, frameworks, and languages you used:

  • Examples: "python", "react", "django", "tensorflow", "postgres", "claude-code", "gpt5", "cursor", "kling"
  • Help other developers find projects using specific tech stacks
  • Show your technical expertise

Adding tags and tools

When creating or editing a drop:

  • Use the dedicated tags and tools fields in the editor
  • Get popular suggestions based on platform usage
  • Or add via frontmatter in your drop content

Frontmatter example

title: My Web App
all_tags: ["web-app", "productivity", "saas"]
all_tools: ["python", "django", "react", "postgres", "claude-code"]
