Media uploads and galleries

22 Aug, 2025

Vibera supports comprehensive media uploads for showcasing your projects with images, videos, and other files.

Supported file types

  • Images: png, jpg, jpeg, tiff, bmp, gif, svg, webp, avif, ico, heic
  • Videos: mp4, webm, mkv
  • Audio: mp3, ogg, wav, opus, flac
  • Documents: pdf, doc, docx, ppt, pptx, xls, xlsx, txt, rtf, epub
  • Fonts: woff, woff2, ttf, otf

Upload limits

  • File size: 10MB maximum per file
  • Per drop: 15 files maximum per drop
  • Total storage: 20,000 files per space (fair use limit)

Uploading media

Upload media through the drop editor:

  • Drag and drop files into the media section
  • Or click to browse and select files
  • Media appears as thumbnails with click-to-insert functionality
  • Images and videos show visual previews

Using media in drops

Once uploaded, media can be inserted into your drop content:

  • Click on media thumbnails to insert into your content
  • Images are automatically formatted for display
  • Videos include playback controls
  • Files are served from fast CDN storage

Media management

Access the media center from your dashboard to:

  • View all uploaded media for your space
  • Upload files not tied to specific drops
  • Manage your media library
  • Monitor your storage usage

Media uploads help you create rich project showcases with screenshots, demo videos, and supporting files.
